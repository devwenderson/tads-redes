# TADS - Redes de computadores

## Úteis
### Python
- `|` - OR Bitwise
- `&` - AND Bitwise
- `^` - XOR Bitwise
- `~` - NOT Bitwise

### Cálculos de redes
- `Endereço de rede`: Máscara AND Endereço do host
- `Broadcast`: Máscara invertida OR Endereço do host
- Endereço de rede é par
- Broadcast é ímpar
